This work studies the security implications of third-party API routers in agentic software development. It covers major coding-agent systems and proposes a router-side injection taxonomy together with a non-intrusive experimental framework for trace capture, replay, malicious injection, and defense evaluation.

The project focuses on:

- modeling router-side supply-chain attacks from explicit response replacement to stealthier distribution-aligned injection,
- building the SIDEL experimental framework,
- constructing a high-quality benchmark dataset for evaluation, and
- quantitatively analyzing the failures of mainstream client-side defenses.
